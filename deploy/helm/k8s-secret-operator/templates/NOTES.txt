The controller is now watching for Secrets with the annotation:

Example usage:
---------------
Create a Secret with the autogenerate annotation:

  apiVersion: v1
  kind: Secret
  metadata:
    name: my-secret
    annotations:
      secgen.gtrfc.com/autogenerate: "password,api-key"
      secgen.gtrfc.com/length: "32"
      secgen.gtrfc.com/type: "string"
  type: Opaque
  data:
    username: YWRtaW4=  # existing value, won't be changed

The controller will automatically generate values for 'password' and 'api-key' fields.

------------------------

View controller logs:
